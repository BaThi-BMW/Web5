# Phân Biệt Website Và Trang Web (Webpage)

Dự án nhỏ này nằm trong chuỗi bài tập thuộc Bài mở đầu thiết kế Web, giúp làm quen và sử dụng thành thạo cấu trúc bảng trong HTML cơ bản.

## 📌 Yêu cầu hoàn thành
* Xây dựng một bảng dữ liệu (`<table>`) gồm 2 cột: **Khái niệm** và **Giải thích**.
* Định nghĩa rõ ràng hai hàng nội dung:
  * **Website:** Hệ thống lớn, tập hợp nhiều trang.
  * **Trang web:** Một trang đơn lẻ, một phần tử thuộc website.
* Sử dụng HTML thuần túy, không lạm dụng CSS phức tạp theo đúng tiêu chí tối giản.

## 🚀 Công nghệ
* HTML5 (Sử dụng các thẻ bảng: `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`).
